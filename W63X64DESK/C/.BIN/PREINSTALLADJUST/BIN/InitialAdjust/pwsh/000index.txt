# Описание файлов в каталоге

Представлены полезные файлы, которые необходимо запускать после
установки операционной системы на жёсткий диск для дальнейшей 
её настройки. Данные скрипты подготавливают систему к активации
и установки админ-пака.

Дальнейшую инструкцию смотри на C:\.BIN\smbshare\000index.txt.md

smbshare-mount.ps1	-- This Script Copy Windows Adjust Files
			 and Shares a Folder with Copied Files
			-- Скрипт копирует файлы, необходимые для 
			настройки Windows, и расшаривает папку 
			с этими файлами
NIT-TinyExclusions.ps1 	-- This Script Add Necessary Tiny 
			Exclusions on Some Directories and Processes
			-- Скрипт добавляет несколько исключений
			Windows Defender на процессы и папки.

Скрипты запускаются в оболочке PowerShell. Порядок запуска: 
- NIT-TinyExclusions.ps1
- smbshare-mount.ps1

Запускать из-под учётной записи локального администратора 
с повышенными привилегиями.

Скрипты не сертифицированы ФСТЭК.
